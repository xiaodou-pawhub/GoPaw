## 🔧 快速开始

### 编译运行

```bash
cd /Users/luoxiaodou/workspace/projects/claw/GoPaw

# 编译
go build ./...

# 生成配置
./gopaw init

# 编辑配置（填入 API Key）
vim config.yaml

# 启动
./gopaw start
```

### 测试

```bash
# 一键端到端测试
./test_e2e.sh

# 手动测试 API
curl http://localhost:8088/health
curl -X POST http://localhost:8088/api/agent/chat \
  -d '{"session_id":"test","content":"你好"}'
```

### 查看数据库

```bash
sqlite3 data/gopaw.db ".tables"
sqlite3 data/gopaw.db "SELECT * FROM messages LIMIT 5;"
```
