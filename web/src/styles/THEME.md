## 🎯 设计理念

### 1. 简洁专业
- 纯白背景营造干净、专业的视觉感受
- 减少视觉干扰，突出内容本身
- 适合长时间使用，不易疲劳

### 2. 层次分明
- 通过微妙的灰度差异区分不同区域
- 边框使用浅灰色，不抢眼但清晰
- 文字颜色分级明确，信息层次清晰

### 3. 强调色点睛
- 珊瑚橙作为强调色，温暖而不张扬
- 仅在关键交互点使用（按钮/链接/选中状态）
- 避免大面积使用，保持克制

## 📋 使用规范

### 背景使用

```css
/* 主应用背景 */
.app { background: var(--bg-app); }

/* 侧边栏/导航 */
.sidebar { background: var(--bg-sidebar); }

/* 卡片/面板 */
.card { background: var(--bg-panel); }

/* 悬浮层/遮罩 */
.overlay { background: var(--bg-overlay); }
```

### 文字使用

```css
/* 标题/重要内容 */
.title { color: var(--text-primary); }

/* 正文/描述 */
.description { color: var(--text-secondary); }

/* 提示/辅助信息 */
.hint { color: var(--text-tertiary); }

/* 禁用状态 */
.disabled { color: var(--text-disabled); }
```

### 边框使用

```css
/* 标准边框 */
.card { border: 1px solid var(--border); }

/* 微妙的分隔 */
.divider { border-top: 1px solid var(--border-subtle); }
```

## 💡 最佳实践

### ✅ 推荐

1. **保持对比度**
   - 文字与背景对比度至少 4.5:1
   - 重要信息使用主文字色

2. **克制使用强调色**
   - 仅在交互元素上使用
   - 避免大面积纯色块

3. **合理使用留白**
   - 元素之间保持足够间距
   - 避免内容过于拥挤

### ❌ 避免

1. **不要混用主题**
   - 不要同时使用深色和浅色元素
   - 保持主题一致性

2. **不要使用纯黑**
   - 避免 `#000000`，使用 `#111827`
   - 纯黑对比度过强，易疲劳

3. **不要过度设计**
   - 避免过多颜色
   - 保持简洁专业

## 🔧 自定义

如需调整强调色，修改 `web/src/styles/globals.css`：

```css
:root {
  /* 改为蓝色强调 */
  --accent:        #3B82F6;
  --accent-hover:  #60A5FA;
  --accent-dim:    rgba(59, 130, 246, 0.1);
  
  /* 或改为绿色强调 */
  --accent:        #10B981;
  --accent-hover:  #34D399;
  --accent-dim:    rgba(16, 185, 129, 0.1);
}
```
